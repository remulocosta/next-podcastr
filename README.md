## :construction_worker: How to run

```
# Clone Repository
$ git clone https://github.com/remulocosta/next-podcastr

# Go to folder
$ cd podcastr

# Install Dependencies
$ yarn

# Run Web Server
$ yarn dev

# Run Server
$ yarn server
```
